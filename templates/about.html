Project Overview

Hand Gesture Recognition is an innovative project that uses computer vision and machine learning to recognize and interpret hand gestures in real-time. This project demonstrates the practical application of artificial intelligence in human-computer interaction.

Key Features:

  • Real-time hand gesture recognition
  • Support for 5 different gestures
  • Web-based interface
  • Text-to-speech capability
  • Gesture history tracking
